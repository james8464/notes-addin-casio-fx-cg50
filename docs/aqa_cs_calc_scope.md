# AQA 7517 CS Calculation Scope

Primary target: AQA A-level Computer Science 7517 Paper 2 calculation questions.

Sources checked:
- AQA 7517 subject content: fundamentals of data representation and Paper 2 theory scope.
- AQA assessment resources and public Paper 2 archives for 2017-2024.
- Recurring past-paper calculation families around number bases, two's complement, floating point, storage, compression, and data rates.

Features for `CSCALC.g3a`:
- binary, denary, hexadecimal conversion
- unsigned ranges and overflow
- two's complement encode/decode/subtraction/range
- fixed-point binary conversion
- floating-point mantissa/exponent decode and encode
- normalisation and closest representable values
- binary addition and shifts
- image, sound, character, file-size, and bit-rate calculations
- compression ratio and percentage saving
- database record/key arithmetic where required

The app should show exam working lines, not just final answers.
