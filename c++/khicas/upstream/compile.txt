Linux amd64 toolchain install to compile KhiCAS:

* Graph 90/Prizm
Compiler and Libc
get casiolocal.tgz, untar it
get libmpfr.so.4, copy it to /usr/local/lib,
add /usr/local/lib to /etc/ld.so.conf and run ldconfig
KhiCAS
get giac90.tgz (1 addin), or giac2.tgz (2 addins) or giacbf.tgz (2 addins with
bigfloat support), untar and run make
You will also need mkg3a (local copy available here)

* Graph 35:
get giac35.tgz, untar it, go to src0 and run make.
