# Structured Query Language (SQL)

## The Select Command

- retrieves data from a table
  - select <attribute>
  - from <table>
  - where <condition>
  - order by <ASC/DESC> -- optional

## The Update Command

- modifies the attributes of an existing entity
- usually uses the primary key to identify the entities to update, but it can also be used with more general conditions
  - update <table>
  - set <attribute> = <value>
  - where <attribute> = <value>

## The Delete Command

- removes entities from a database
  - delete
  - from <table>
  - where <condition>

## The Insert Command

- adds new records to an existing table
  - insert into <table>
  - values <values>

## Wildcards

- used to specify any possible value
- eg rather than selecting a specific attribute in the select command, you could use a wildcard to return all attributes
- wildcards are notated using an asterix (ie a * symbol)

### Example

  - select *
  - from <Cars>
  - where (price > 4000)
  - the example above would return all of the details of each car that costs more than 4000

## The Create Command

- used to create new database tables
- specifies the name of the new table, its attributes (columns) and data types of each attribute, and identifiers (eg primary key)
  - create table <tableName> (
      - <attribute1> <dataType1>,
      - <attribute2> <dataType2>,
      - <attribute3> <dataType3>,
      - primary key (<attribute1>,<attribute2>) -- shows how to define a table with a composite PK
      - )
- create table syntax and marking points (remember you need a comma after each line)

### SQL Data Types

| data type | SQL | description |
| --- | --- | --- |
| fixed length string | char(size) | a string with a specified num of characters |
| variable length string | varchar(size) | a string with any num of characters up to the specified num |
| integer | int(size) | a whole number stored using a specified num of bits |
| number with fractional part | float(size, precision) | number stored using specified num of bits with a specified num of digits after the decimal point |
| date | date | a date in the format: YYYY-MM-DD |
| date and time | datetime | a date and time combined in the format:; YYYY-MM-DD HH:MM:SS |
| time | time | a time in the format HH:MM:SS |
| year | year | a year in one of the two formats below: yy or yyyy |
