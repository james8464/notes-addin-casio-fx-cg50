# Functional language programs

- The following symbols can be used, same as python...
- +
- -
- *
- /

### Arithmetic Examples

ghci> 2 + 15
17
ghci> 49 * 100
4900
ghci> 1892 - 1472
420
ghci> 5 / 2
2.5
ghci>

- Same as in python.
- Used to make the precedence explicit or change it
- Also makes the code more readable for humans

### Bracket Precedence Examples

ghci> (50 * 100) - 4999
1
ghci> 50 * 100 - 4999
1
ghci> 50 * (100 - 4999)
-244950

### The Available Operations Are

- && meaning and
- || meaning or
- NOT which negates a True or a False

### Boolean Examples

ghci> True && False
False
ghci> True && True
True
ghci> False || True
True
ghci> not False
True
ghci> not (True && True)
False
- The following equality comparisons can be used in addition:
- '==' which means is equal to
- '/=' which means is not equal to

### Equality Examples

ghci> 5 == 5
True
ghci> 1 == 0
False
ghci> 5 /= 5
False
ghci> 5 /= 4
True
ghci> "hello" == "hello"
True

### 'succ'

- Takes anything that has a defined successor
- returns that successor

### Succ Example

ghci> succ 8
9

### 'min' and 'max'

- Takes two inputs
- 'min' returns the one that is lesser
- 'max' returns the one that is greater

### Min and Max Example

ghci> min 9 10
9
ghci> max 100 101
101
