# First-class object

### Define the Term 'first-class Object' (ppq)

- the name given to a type of object which can be passed in or returned from a function

### A First-class Object Is an Object/value Which May

- Appear in expressions
- Be assigned to a variable
- Be assigned as arguments
- Be returned in function calls

### Examples

- Integers
- Floating-point values
- Characters
- Strings (list of characters)
- Booleans
- Functions

### Note

      - If a question asks for examples of first class object, you cant just say 'data types'... be specific, avoid generalising
