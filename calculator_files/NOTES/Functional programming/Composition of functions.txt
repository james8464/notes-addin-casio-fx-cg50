# Composition of functions

- definition: the process of combining a series of functions into a single function

- note: its exactly the same as function composition in maths

- functional composition combines two functions to get a new function.

- eg if f : A -> B, and g : b -> c, then:

    g o f

- is called the composition of g and f, and is a function whose domain is a and co-domain is c.

- composition is possible because the co-domain of f is the same as the domain of g.

## Example

  - f(x) = 2x + 3
  - g(x) = 4x
  - g(f(x)) = 4( 2x + 3 )
