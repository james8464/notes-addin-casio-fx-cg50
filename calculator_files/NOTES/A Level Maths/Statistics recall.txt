# statistics recall

## sampling

- population means the whole group being studied
- sample means the selected part of the population
- census uses every member of the population
- sample is cheaper and faster than a census
- sampling frame is a list of possible sampling units
- random sampling reduces bias
- opportunity sampling is quick but often biased
- systematic sampling uses a random start and fixed interval
- stratified sampling represents subgroups in proportion to their size
- quota sampling fills categories until quotas are met



## data types

- qualitative data is non-numerical
- quantitative data is numerical
- discrete data takes separate countable values
- continuous data can take any value in an interval
- primary data is collected first hand
- secondary data already exists
- bivariate data has paired values for two variables



## measures of location

- mean uses every value but is affected by outliers
- median is resistant to outliers
- mode is useful for qualitative or discrete data
- if data is skewed, median is often more representative



## measures of spread

- range = max - min
- IQR = upper quartile - lower quartile
- IQR is resistant to outliers
- standard deviation measures spread around the mean
- larger standard deviation means more variation



## outliers

- common rule is below Q1 - 1.5IQR or above Q3 + 1.5IQR
- outliers may be genuine extreme values
- outliers may be recording errors
- never remove an outlier without a reason
- use context to decide whether an outlier is suspicious



## histograms

- used for continuous grouped data
- area represents frequency
- frequency density = frequency / class width
- unequal class widths need frequency density
- total frequency is found from total area



## box plots

- show minimum, Q1, median, Q3, maximum
- compare medians for typical value
- compare IQR or range for spread
- check skew from median position and whisker lengths
- show outliers separately if required



## coding data

- coding changes data to simplify calculation
- if y = (x - a) / b, then x = by + a
- mean transforms using the same linear coding
- standard deviation only scales by b, it is not affected by adding a
- variance scales by b^2



## probability

- mutually exclusive events cannot both happen
- independent events do not affect each other
- P(A or B) = P(A) + P(B) - P(A and B)
- if independent, P(A and B) = P(A)P(B)
- conditional probability means probability given another event has happened
- tree diagrams are useful for successive events



## binomial distribution

- fixed number of trials
- two outcomes per trial
- constant probability of success
- trials are independent
- X ~ B(n, p)
- use binomial when counting successes
- mean = np
- variance = np(1 - p)



## normal distribution

- continuous bell shaped model
- symmetrical about the mean
- total area under curve is 1
- X ~ N(mu, sigma^2)
- standardise using Z = (X - mu) / sigma
- use continuity correction when approximating a discrete distribution
- normal model may be poor for skewed or bounded data



## hypothesis tests

- H0 is the default claim
- H1 is the alternative claim
- significance level is the probability of wrongly rejecting H0
- one tailed test looks for increase or decrease
- two tailed test looks for difference
- reject H0 if p-value is less than significance level
- reject H0 if test statistic falls in critical region
- conclusion must be written in context
- do not say H0 is proved



## correlation and regression

- PMCC measures strength and direction of linear correlation
- PMCC is between -1 and 1
- PMCC near 1 means strong positive linear correlation
- PMCC near -1 means strong negative linear correlation
- PMCC near 0 means little linear correlation
- correlation does not prove causation
- regression line of y on x predicts y from x
- interpolation is inside the data range
- extrapolation is outside the data range and less reliable
- outliers can strongly affect correlation and regression



## exam wording

- "state" needs a short answer
- "interpret" needs context
- "comment" needs comparison or judgement
- "give a reason" needs a statistical reason, not just a calculation
- "in context" means mention the variable and situation
