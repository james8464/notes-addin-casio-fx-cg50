# Absolute and relative errors

## Absolute Errors

- The actual amount by which a value is inaccurate
- Calculated by finding the difference between the given value and the actual value

### Example

## Relative Errors

- A measure of uncertainty in a given value compared to the actual value
- Relative to the size of the given value

- Larger actual value = smaller relative error

## Fixed Point

| Advantages | Disadvantages |
| --- | --- |
| Numbers are represented exactly | Limited range of numbers that can be represented |
| Faster processing times compared to floating point numbers |  |

## Floating Point

| Advantages | Disadvantages |
| --- | --- |
| Allows a far greater range of numbers for the same number of bits | Rounds off large numbers |
| Can represent very large and very small numbers | Slower processing times |
