# Vector graphics versus bitmapped graphics

## Comparison

|  | Bitmapped (raster) graphics | Vector graphics |
| --- | --- | --- |
| Advantage 1 | Very realistic images (good for photos) | Can be scaled without loss of quality (resolution independent) |
| Advantage 2 | High detail and subtle colour variation | Smaller file sizes |
| Advantage 3 | Easy to edit at pixel level | Sharp, clean lines |
| Advantage 4 | Widely supported formats (JPEG, PNG) | Easy to edit objects (move, resize shapes) |
| Advantage 5 | Simple to capture (cameras) | Ideal for logos, icons, diagrams |
| Advantage 6 | No processing needed to render (just display pixels) | Good for printing at any size |
| Advantage 7 |  | Lower memory usage when stored |
| Disadvantage 1 | Large file sizes (especially high resolution) | Not suitable for realistic images/photos |
| Disadvantage 2 | Pixelation when enlarged | Cannot represent fine detail or complex textures well |
| Disadvantage 3 | Quality depends on resolution | More processing required to render |
| Disadvantage 4 | Not easily scalable | Limited colour realism compared to bitmaps |
| Disadvantage 5 | Editing large images can be slow |  |

## Uses for Bitmap Images

- Real-world photography
- Photos of people

## Uses for Vector Graphics

- Company logos
- Diagrams
- Charts
- icons
