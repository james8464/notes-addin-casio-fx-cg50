# Number systems

## Natural Numbers (N)

- all the positive integers and zero
- N = {0, 1, 2, 3, 4, ...}
- ie 1,2,3,4,5...
- used for counting

## Integer Numbers (Z)

- both positive and negative numbers and zero.
- Z = { ..., -3, -2, -1, 0, 1, 2, 3, ... }

## Rational Numbers (Q)

- any number that can be written as a fraction (where denominator is not zero)
- these include the positive and negative integers since they can be written as a fraction over 1

## Irrational Numbers

- any number which cannot be written as a fraction
- examples
  - sqrt(2)
  - pi
  - e

## Real Numbers (R)

- any number which can be represented in the real world
- includes the rationals, irrationals, integers, and natural numbers
- used for measurements
- real numbers are used for measurement (since most values that are measured are decimal or in terms of irrational numbers)

## Ordinal Numbers

- used to describe the position of an item in a set
- examples
  - 1st
  - 2nd
  - 3rd
