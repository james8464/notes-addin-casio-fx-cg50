# Numbers with a fractional part

## Fixed Point Binary

- a specified num of bits are placed before a binary point
- the remaining bits fall behind the binary point
- before the binary point, placeholders still work as normal (ie 2^0, 2^1, 2^2 .. etc)
- after the binary point, placeholders work like following:
  - 2^-1, 2^-2, 2^-3 ... etc

## Floating Point Binary

- contains a mantissa and an exponent
- both mantissa and exponent will be represented using two's complement signed binary (see 4.5.4.3)

### Converting Floating Point Binary -> Decimal

- convert exponent to decimal
- imagine there was a binary point between the first and second digits of the mantissa
- move that imaginary binary point the number of positions specified by the exponent
- treat this mantissa as a fixed point binary number and convert it to decimal

### Example

### Converting Decimal to Floating Point Binary

- convert decimal number to fixed point binary
- move the binary point until the first two digits of the binary value are different
  - if the point moves left, the mantissa would be positive
  - if the point moves right, the mantissa would be negative
