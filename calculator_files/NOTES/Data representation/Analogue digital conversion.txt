# Analogue digital conversion

## How does ADC work?

- ADC converts analogue data into digital data
- ADC takes samples of the analogue signal at regular intervals
  - frequency of sampling = sampling rate
  - measured in Hz
  - higher sampling rate = better reproduction of the original signal
  - lower sampling rate = less accurate representation
- samples are quantised
  - amplitude is approximated/rounded to the nearest available value
  - introduces quantisation error
- each sample is encoded as a binary value
- binary values can then be stored, processed or transmitted by a computer


## Uses of ADC

- used whenever a computer receives analogue data
- many sensors output analogue signals:
  - microphones
  - temperature sensors
  - light sensors
  - pressure sensors
- ADC allows computers to process these signals



- quality of the digital representation depends on:
  - sampling rate
  - sample resolution

- sample resolution = number of bits used per sample
  - higher sample resolution = greater accuracy
  - higher sample resolution = more possible amplitude values
  - lower sample resolution = greater quantisation error

- Nyquist Theorem:
  - sampling rate must be at least twice the highest frequency present in the analogue signal
  - prevents loss of information
  - prevents aliasing



## How does DAC work?

- DAC converts digital data into analogue data
- DAC takes binary data as input
- each binary value is converted into a corresponding analogue voltage level
- voltage levels are output in sequence
- initial output is a stepped signal
- signal is smoothed (eg using a low-pass filter)
- produces a continuous analogue waveform


## Uses of DAC

- used whenever a computer outputs analogue signals
- most common use:
  - converting digital audio into analogue audio
- used in:
  - speakers
  - headphones
  - sound cards
  - audio playback devices
