# MIDI

## What Is MIDI

- musical instrument digital interface
- a protocol that allows computers, musical instruments and other hardware to communicate
- a communication standard that allows digital music gear to speak the same language
- stores instructions for playing music, rather than storing the actual sound wave
- does not sample audio like digital sound recording
- can be used to control synthesisers, keyboards, sound cards and music software
- the same MIDI file can sound different depending on the device/software used to play it

## How Does MIDI Work

- music is represented as a sequence of MIDI (event) messages
- each event message tells a device what musical action to perform
- event messages are processed in time order
- examples of data that might be included in a message:
  - channel
  - note on/off
  - pitch/frequency/note number
  - volume
  - duration
  - velocity
  - key pressure
  - timbre
  - instrument
  - pedal effects
  - pitch bend
  - note envelope
- event messages essentially represent an event in a piece of music (think of it as a series of instructions which could be used to recreate a piece of music)
- note on starts a note
- note off stops a note
- velocity can represent how hard a key is pressed
- channel allows different instruments/parts to be controlled separately


## Advantages of Using MIDI

- more compact representation
- much smaller file size than sampled audio
- easy to modify/edit notes
- individual notes, instruments, tempo and timing can be changed
- simple method to compose algorithmically
- musical score can be generated directly from a MIDI file
- no data lost about musical notes (technically lossless)
- no background noise recorded
- MIDI files are often smaller in size as only instructions to recreate each sound are stored
- useful for composing, editing and controlling electronic instruments


## Disadvantages of Using MIDI

- not suitable for recording real sound/audio
- hard to accurately replicate expressive, natural sounds (eg human voice or live instruments)
- requires compatible software/hardware to play properly
- can sound artificial/computer-generated
- limited to musical note data (can't store full audio effects like reverb, distortion or nuanced timbre in detail)
- playback quality depends on the synthesiser/sound bank used
- cannot store vocals, speech or environmental sounds directly
- less suitable when the exact original performance or sound must be preserved
