# Rounding errors

- Eg 1/3 (decimal = 0.3333333...) cant be EXACTLY represented, only approximately

- can never allocate enough bits to store that number since it never ends

## Define the Term rounding error

- Define the term 'rounding error' (re-worded PPQ):
- A value cannot be represented exactly in the available number of bits 
- A value cannot be represented exactly in the available number of bits / digits
- The stored value is rounded/truncated to the nearest representable value, creating a difference between the actual value and the stored value
