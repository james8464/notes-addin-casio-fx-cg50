# Communication basics

## Definitions

- baud rate
  - number of signal changes per second
- bit rate
  - number of bits transferred per second
  - bit rate would be higher than baud rate when the number of bits transferred per signal is more than one
- bandwidth
  - the range of frequencies within a given band used for transmission of a signal
- latency
  - time delay between the moment transmission is initiated and the moment its first effect begins
  - usually increases with distance
- protocol
  - a set of rules relating to the communication between devices


## Equation for bit rate

- Bit rate = baud rate multiplied by the number of bits per signal




## Relationship between bit rate and bandwidth

- directly proportional
