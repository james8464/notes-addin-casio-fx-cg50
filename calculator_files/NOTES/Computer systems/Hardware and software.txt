# Hardware and software

## Hardware

- physical components of a computer system
- if you can touch it, it is hardware




## Software

- programs/code/instructions executed to perform a task
- name given to programs/code/instructions that are executed in order to perform a task

### Relationship with Hardware

- software runs on hardware
- hardware executes software instructions



### Application Software

- programs that complete a specific task for the user
- examples:
  - word processors
  - web browsers
  - spreadsheet software



### System Software

- operates, controls and maintains the computer and its components
- types:
  - operating system
  - utility programs
  - library programs
  - translators



## Operating System (OS)

### Main Purpose

- hides the complexities of the hardware from the user
- manages and allocates computer resources

### Processor Management

- allocates processors/cores to processes
- schedules processes
- decides which process runs when

### Memory Management

- allocates memory/RAM to processes
- manages movement of data into and out of RAM
- supports virtual memory
- ensures processes only access allocated memory

### I/O Device Management

- allocates I/O devices to processes
- manages communication between processes and I/O devices
- supports installation of new devices/drivers

### File Management

- allocates storage space to files
- manages files and directories
- determines where files are stored
- manages file allocation structures

### Application Management

- allows installation of new software
- manages software updates

### Security Management

- authenticates users
- manages file permissions
- prevents unauthorised access to resources

### Other Functions

- handles interrupts
- manages power consumption




## Utility Programs

### Main Purpose

- perform housekeeping tasks in a computer system
- non essential programs
- examples:
  - encryption software
  - defragmentation software
  - data compression software
  - backup software
  - antivirus software



## Library Programs

### Main Purpose

- contain useful functions frequently used by programs
- reduce development time
- can be imported and reused by programmers



## Translators

### Main Purpose

- translate between different types of language
- convert source code into a form that can be executed
- types:
  - compiler
  - interpreter
  - assembler
