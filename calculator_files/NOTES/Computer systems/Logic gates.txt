# Logic gates

## Half Adder

### Characteristics

- two inputs
- two outputs
- two logic gates (and & XOR)
- can be used to add two boolean values




## Full Adder

### Characteristics

- three inputs
- two outputs
- can input 2 boolean values and a carry bit from a previous, less significant operation



## D-type Flip-Flop

- can be used as a memory unit for storing one bit
- has 2 inputs: data and a clock signal/pulse
- produces 1 output: the value of the stored bit


### How it works

- clock signal is generated by the computer
  - alternates between 0 and 1
- value of the stored bit is set to the value of the data input with each change of the clock signal
