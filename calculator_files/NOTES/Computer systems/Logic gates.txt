# Logic gates

## Basic Gates

- logic gates take binary inputs and produce a binary output
- used to build circuits inside the CPU and other digital systems
- output depends only on the current input values
- truth tables show every possible input combination and output




## Half Adder

### Characteristics

- adds two binary digits
- inputs:
  - a
  - b
- outputs:
  - sum
  - carry
- sum = a XOR b
- carry = a AND b
- no carry-in input
- useful for adding the least significant bits in a binary addition circuit
- cannot handle a carry from a previous column




## Full Adder

### Characteristics

- adds two binary digits and a carry-in bit
- inputs:
  - a
  - b
  - carry-in
- outputs:
  - sum
  - carry-out
- sum = a XOR b XOR carry-in
- carry-out = 1 if at least two inputs are 1
- can be chained together to add multi-bit binary numbers
- carry-out from one full adder becomes carry-in for the next more significant bit



## D-type Flip-Flop

- can be used as a memory unit for storing one bit
- has 2 main inputs:
  - data input
  - clock signal/pulse
- produces an output showing the stored bit
- often called a data flip-flop or delay flip-flop
- used in registers, counters and memory circuits


### How it works

- clock signal is generated by the computer
  - alternates between 0 and 1
- when triggered by the clock, the stored bit is set to the data input
- between clock pulses, the output stays the same
- this allows a circuit to store a stable value until the next clock signal
