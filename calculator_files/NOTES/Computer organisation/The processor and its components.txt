# The processor and its components

## Major Processor Components

### ALU

- arithmetic logic unit
- performs arithmetic and logical operations on data
- does not do the execution (execution is performed by the processor)




### CU

- control unit
- controls fetch/load/store operations
- decodes instructions
- manages the execution of instructions
- runs the FE cycle
- sends control signals to other components
- stores the state of the processor when an interrupt occurs




### Clock

- generates signals
- synchronises the operation of the processor and the movement of data around the other components of the computer
- a CPU cannot perform operations faster than the clock cycle




### General-purpose Registers

- store values that need to be accessed frequently
- store values that instructions can/will use as operands
- programmer determines the role of each register

## Registers

- high-speed memory locations
- located in the processor/CPU
- used by the processor to store small amounts of data needed during processing




### Special Purpose Registers

PC:

- program counter
- stores the address of the next instruction

MAR:

- memory address register
- stores the address of the instruction currently being fetched

MBR / MDR:

- memory buffer register
- stores the contents of the memory address in the MAR

CIR:

- current instruction register
- stores the instruction currently being decoded/executed

ACC:

- accumulator
- stores the result of a calculation once the instruction is executed in the ALU

SR:

- status register
- contains information about the result of the last instruction
- contents are used to affect the behaviour of the next instruction
- stores bits/flags which affect the operation of the control unit

IR:

- interrupt register
- stores signals that have been received by the processor from other components
