xform(expr,target)

Args:
  expr, target

Required:
  start expression, target expression/form

Optional:
  None.

Does:
  Transforms one expression into a target form using supported algebra, trig, log, factorisation, expansion, and coefficient-matching steps.

Examples:
  xform(1+tan(x)^2,sec(x)^2); xform(log(a,x),ln(x)/ln(a))

F2:
  xform(1+tan(x)^2,sec(x)^2)

F3:
  xform(log(a,x),ln(x)/ln(a))

Keys:
  F1 inserts the command.
  F2 inserts the first example.
  F3 inserts the second example when available.

Working:
  Tries identities, simplify/expand/factor/collect, log laws, trig identities, and constants matching before falling back.

Tip:
  Use brackets around compound numerators, denominators, powers, and substitutions for readable exam-style output.
