trigcos(expr)

Args:
  expr

Required:
  expr

Optional:
  None.

Does:
  Rewrites trig expressions using cosine where useful.

Examples:
  trigcos(sin(x)^2); trigcos(tan(x))

F2:
  trigcos(sin(x)^2)

F3:
  trigcos(tan(x))

Keys:
  F1 inserts the command.
  F2 inserts the first example.
  F3 inserts the second example when available.

Working:
  Usually final-answer only unless used inside a longer routed solution.

Tip:
  Use brackets around compound numerators, denominators, powers, and substitutions for readable exam-style output.
