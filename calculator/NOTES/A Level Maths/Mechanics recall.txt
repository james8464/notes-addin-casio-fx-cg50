# mechanics recall

## modelling assumptions

- particle means size and rotation are ignored
- rod means length matters but thickness is ignored
- lamina means flat object with negligible thickness
- light means mass is negligible
- inextensible string means same acceleration throughout the string
- smooth surface means no friction
- rough surface means friction may act
- uniform object means mass is evenly distributed
- centre of mass is at geometric centre for a uniform body
- air resistance is ignored unless stated
- g is usually 9.8 m s^-2 unless stated otherwise



## units

- displacement uses m
- velocity uses m s^-1
- acceleration uses m s^-2
- force uses N
- mass uses kg
- weight = mg
- moment uses N m
- power uses W
- impulse uses N s



## vectors

- displacement is vector
- velocity is vector
- acceleration is vector
- force is vector
- speed is scalar
- distance is scalar
- mass is scalar
- time is scalar



## SUVAT

- use SUVAT only when acceleration is constant
- s = displacement
- u = initial velocity
- v = final velocity
- a = acceleration
- t = time
- choose positive direction before writing equations
- signs matter for displacement, velocity and acceleration
- vertical motion usually has acceleration -g if upwards is positive



## displacement time graphs

- gradient gives velocity
- steeper gradient means greater speed
- horizontal section means object is stationary
- curved graph means velocity is changing



## velocity time graphs

- gradient gives acceleration
- area under graph gives displacement
- area below time axis gives negative displacement
- speed is magnitude of velocity
- distance travelled counts areas as positive



## acceleration

- acceleration is rate of change of velocity
- if v is given as a function of t, then a = dv/dt
- if x is given as a function of t, then v = dx/dt
- displacement is found by integrating velocity
- velocity is found by integrating acceleration
- use initial conditions to find constants



## projectiles

- split motion into horizontal and vertical components
- horizontal acceleration is 0
- vertical acceleration is -g if upwards is positive
- time is shared by horizontal and vertical motion
- at greatest height, vertical velocity is 0
- if landing at same height, time up equals time down
- range means horizontal distance travelled



## forces

- draw a clear force diagram first
- resultant force = ma
- forces in equilibrium have resultant 0
- tension acts along a string away from the object
- normal reaction acts perpendicular to the surface
- weight acts vertically downwards
- thrust or driving force acts in direction of motion
- resistance usually acts against motion



## connected particles

- particles joined by light inextensible string have same acceleration
- tension is the same throughout a light string over a smooth pulley
- write F = ma separately for each particle
- choose consistent positive directions
- internal tension often cancels if equations are added



## inclined planes

- resolve parallel and perpendicular to the plane
- weight component down the plane is mg sin theta
- weight component into the plane is mg cos theta
- normal reaction is usually mg cos theta unless extra forces act
- acceleration along the plane comes from resultant parallel force



## friction

- friction opposes actual or possible motion
- limiting friction is F = mu R
- if moving, friction is normally mu R
- if at rest, friction is at most mu R
- use F <= mu R for static equilibrium
- coefficient of friction has no units



## moments

- moment = force x perpendicular distance from pivot
- clockwise moments can be balanced with anticlockwise moments
- taking moments about a point removes forces through that point
- use moments for rods, ladders and rigid bodies
- equilibrium needs resultant force 0 and resultant moment 0



## common traps

- do not use speed when velocity sign matters
- do not use SUVAT if acceleration is not constant
- do not forget weight acts vertically, not down a slope
- do not assume friction is limiting unless stated or implied
- do not forget units in final answers
- do not round too early
